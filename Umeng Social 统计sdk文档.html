Umeng Social 统计SDK 文档

首先初始化一个UMSocialStatistic对象，可以改变对象的identifier和cuid属性，然后调用评论、分享、喜欢等方法进行统计。

UMSocialStatistic 属性

identifier

标识应用内不同的资源

@property (nonatomic, copy) NSString *identifier;

cuid

应用自定义的用户id

@property (nonatomic, copy) NSString *cuid;

UMSocialStatistic 方法

setAppKey

设置sdk使用的友盟appkey

+ (void)setAppKey:(NSString *)appKey;

parameters
appKey
友盟appkey

openLog

设置是否显示log

+ (void)openLog:(BOOL)openLog;

parameters
openLog
是否把log显示出来

initWithIdentifier:cuid:;

初始化方法

- (id)initWithIdentifier:(NSString *)identifier cuid:(NSString *)cuid;

parameters
identifier
标识不同统计资源的标示符

cuid
开发者自定义的用户id

setUMSocialDelegate:;

设置回调对象

- (void)setUMSocialDelegate:(id <UMSocialDelegate>)delegate;

parameters
delegate
实现回调协议<UMSocialDelegate>的对象

postCommentWithContent:location: shareToSNSWithUsid:;

发送评论统计

- (void)postCommentWithContent:(NSString *)content location:(CLLocation *)location shareToSNSWithUsid:(NSDictionary *)shareToSNS;

parameters
content
评论内容

location
评论地理位置

shareToSNS
评论并分享的平台，以分享到的平台为key，uid为value。平台名称分别为“sina”、“tencent”、“renren”、“douban”

postSNSWithType: usid: content: weiboId:;

分享完成之后调用此方法，发送分享内容统计

- (void)postSNSWithType:(UMShareToType)umShareType usid:(NSString *)usid content:(NSString *)content weiboId:(NSString *)wid;

parameters
umShareType
分享平台

usid
分享用户的usid

content
分享内容

wid 微博id号

addOrMinusLikeNumber;

添加喜欢，如果再此发送就是取消喜欢 - (void)addOrMinusLikeNumber;

UMSocialDelegate 回调方法

didFinishGetUMSocialResponse;

调用统计方法完成之后实现此回调方法获得发送成功与否

-(void)didFinishGetUMSocialResponse:(UMSResponseEntity *)response;

parameters
response
结果状态对象
